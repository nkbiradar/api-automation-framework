# REST API Automation Framework

A lightweight REST API automation framework built using Python and pytest to validate
API functionality through automated tests and CI integration.

---

## Tech Stack
- Python
- pytest
- requests
- GitHub Actions (CI)

---

## Features
- Automated REST API testing
- Positive and negative test scenarios
- Data-driven testing using JSON files
- Reusable API client utilities
- CI-based test execution using GitHub Actions
